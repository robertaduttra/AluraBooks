# AluraBooks

Criação de uma página inicial da AluraBooks, totalmente responsivo.

Disponível na versão Web, tablet e mobile.

 Tecnologias Utilizadas:
- HTML5
- CSS3
- JavaScript (SwiperAPI)
  
Deploy em: https://robertaduttra.github.io/AluraBooks/
